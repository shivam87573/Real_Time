# 💬 Real-Time Chat Application

A modern Real-Time Chat Application built with **React, Vite, Node.js, Express, and Socket.IO**. This application enables users to chat instantly using WebSockets with a fast and responsive interface.

---

## 🚀 Features

- ⚡ Real-time messaging using Socket.IO
- 👥 Multiple users can chat simultaneously
- 🔄 Instant message updates
- 📱 Responsive UI
- 🎨 Modern React + Tailwind CSS interface
- 🔌 Node.js & Express backend
- 🌐 Cross-platform support

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO
- UUID
- CORS
- Dotenv
